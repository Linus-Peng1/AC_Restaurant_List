# Restaurant_List

<br>

此專案提供使用者瀏覽、搜尋特定的餐廳，並可以點擊餐廳瀏覽更詳細的資訊，如:餐廳類別、地址、評分、描述等。

## 建置環境

<br>

* node.js : 10.15.0
* express: 4.17.1
* express-handlebars: 5.3.2

<br>

## 安裝流程

<br>

1. 開啟終端機(terminal)，利用 git clone 將專案下載至本機
```
git clone https://github.com/Linus-Peng1/AC_Restaurant_List.git
```
2. 進入存放此專案資料夾
```
cd restaurant-list
```
3. 安裝套件
```
npm install
```
4. 啟動網頁伺服器
```
npm run dev
```
